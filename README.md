# Movie Buff
Movie Buff is a movie-themed quiz game inspired by Jeopardy! 

## Getting Started  
Click [here](https://movie-buff-unit-1-project.surge.sh/) to play.  

### Instructions
Select a question and enter your guess. Earn over 7500 points to win.
